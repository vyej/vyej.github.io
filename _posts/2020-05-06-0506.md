2020年4月的閱讀，

2020-04-01 尼采如何克服历史主义 : 尼采《扎拉图斯特拉如是说》讲疏 [美]列奥·施特劳斯 讲疏、[美]理查德·维克利 整理 / 马勇 / 华东师范大学出版社 / 2019-8 / 98.00元 好吧，還是施特勞斯。

2020-04-05 叙拉古的雅典异乡人 : 柏拉图《书简七》探幽 彭磊 选编 / 王师 马涛红 / 华夏出版社 / 2009-12 / 29.00元  關於書簡7來着，好吧，柏拉圖。

2020-04-06 修辞、政治与哲学 : 柏拉图《高尔吉亚》讲疏 [美] 列奥·施特劳斯（Leo Strauss） / 李致远 / 华东师范大学出版社 / 2017-9 / 138.00元  其實應該也是終於看完。

2020-04-07 卢梭的榜样人生 : 作为政治哲学的《忏悔录》 【美】克里斯托弗·凯利 / 黄群 等 / 华夏出版社 / 2009-04 / 30.00元 不是很好看？

2020-04-07 阿维斯塔 贾利尔·杜斯特哈赫 / 元文琪 / 商务印书馆 / 2011-7 / 42.00元  其實只是節選本。

2020-04-09 启蒙运动三书——哲学书简 [法]伏尔泰 / 李筱希 / 北京理工大学出版社 / 2018-9 / 52  只是沒有想清楚，尼采怎麼看待伏爾泰。

2020-04-09 咖啡瘾史 : 一场穿越800年的咖啡冒险 [美] 斯图尔德·李·艾伦 / 简瑞宏 / 广东人民出版社 / 2018-8-1 / 59.00元 其實只是很私人的旅行。

2020-04-09 实用主义 威廉·詹姆士 / 李步楼 / 商务印书馆 / 2011-7 / 20.00元  其實還是實證主義，2.0吧。

2020-04-09 自我的超越性 (法) 萨特 / 杜小真 / 商务印书馆 / 2011-5 / 18.00元  其實只是一篇論文，要成爲一本書不得不加了很多說明和存在與虛無的章節。

2020-04-10 哲学的自传 : 卢梭的《孤独漫步者的遐思》 [美] 迈克尔·戴维斯 / 曹聪、刘振 / 华夏出版社 / 2011-3 / 33.00元 總是感覺痕跡以前就看過，可能是一直沒有看完。

2020-04-12 剑桥美国经济史（第二卷） : 漫长的19世纪 斯坦利·L·恩格尔曼等 / 王珏、李淑清 / 中国人民大学出版社 / 2008-07-21 / 88.00元 好吧，確實不錯！

2020-04-13 尼采反卢梭 : 尼采的道德政治思想研究 凯斯.安塞尔-皮尔逊 / 宗成河、孙磊、熊文驰 / 华夏出版社 / 2005年7月1日 / 29.00元 好像也沒有看出很多什麼來。

2020-04-23 希腊化世界中的犹太人 : 斐洛思想引论 [英] 罗纳尔德·威廉逊 / 徐开来、林庆华 / 华夏出版社 / 2003-8 / 36.00元 這個倒是不錯，不夠貌似看的時間拖長了一些。

2020-04-28 赫耳墨斯秘籍 托名赫尔墨斯 / 肖霄 / 华东师范大学出版社 / 2019-7 / 88.00元 其實就是鍊金術士的書吧。

2020-04-28 不可言说的言说 : 我们时代的上帝问题 H.奥特 / 林克、趙勇 / 三联书店 / 1997-01 / 8.20  當代的神學書，貌似其實也很讀者風。

總算，4月也勉強看完15本書。只是如果按照自己的要求，這個差了太多。
