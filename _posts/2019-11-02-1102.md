貌似，10月確實就是淪陷了。幾乎沒有看完什麼書。國慶放假可以作爲藉口麼，還是沒有加班小一個月。不是還有放假麼。這都是在說什麼呢。

10月其實就讀了三本書……

2019-10-07 解释的冲突 : 解释学文集 [法] 保罗·利科 / 莫伟民 / 商务印书馆 / 2017-12 / 68.00  其實好像利科也是那種材料型而不是熱情型的，哲學教授？

2019-10-12 现象学的心理学 : 1925年夏季学期讲稿 [德] 埃德蒙德·古斯塔夫·阿尔布雷希特·胡塞尔 / 游淙祺 / 商务印书馆 / 2017-7 / 69.00  幾乎還是飛機上看完來着。其實胡塞爾很多時候貌似比海德格爾還激進些吧，然後其實相互影響也是很明顯來着。

2019-10-28 生命政治的诞生 : 法兰西学院课程系列：1978-1979 [法]米歇尔·福柯 / 莫伟民、赵伟 / 上海人民出版社 / 2018-11 / 68 說起來，這應該算是很重要的福柯文獻吧，比如一直糾結的，福柯是如何看待韋伯是個問題。當然，福柯甚至還可以評論一下弗里德曼和加里貝克，哈耶克或者米塞斯應該也看過。比較也是晚輩。

然後就沒有了。雖然貌似唯一的進展是看完了留存多年的紙牌屋第5季，然後發現還有一個評分很低的第6季。說起來，其實第5季也就是崩了吧。
