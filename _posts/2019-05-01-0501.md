說起來，其實也算繼續沒有方向和目標——其實更準確地應該叫沒有熱情和渴望？

4月也算勉強看完15本書，好吧，最有益的閱讀其實應該卻是這個公司金融，雖然也還是有些胡亂快進的。

不過總體而言確實很有收穫。

2019-04-08 影子银行内幕 : 下一个次贷危机的源头? 张化桥 / 黎木白 / 机械工业出版社 / 2013-10 / 39.00  雖然貌似作者重點在吐槽，不過確實還是很有料的，雖然這並沒有直接參與，不過懂行的人，說其實都還是說在點上來着？

2019-04-10 红色资本 : 中国的非凡崛起与脆弱的金融基础 [美]卡尔·沃尔特(Carl E. Walter)、[美]弗雷泽·豪伊(Fraser J. Howie) / 祝捷、刘骏 / 东方出版中心 / 2013-8-1 / 38.00元  雖然不免懷有些國外的叵測，但是有些事情說破之後其實道理也挺淺顯來着？

2019-04-10 希罗多德历史（套装上下册） [古希腊] 希罗多德 / 王以铸 / 商务印书馆 / 2011-7 / 89.00元  中間有一大部分看得有些迷糊。

2019-04-15 农村信用社贷款五级分类操作手册  张功平 / 第1版 (2004年8月1日) / 2004-8 / 16.00 湊數讀的來着。

2019-04-18 奥义书 黄宝生 / 商务印书馆 / 2011-5 / 45.00元  什麼大森林奧義貌似比較神奇，其他好像也還好了。

2019-04-21 善恶的彼岸 [德] 弗里德里希·尼采 / 朱泱 / 团结出版社 / 2001-9 / 15.00元 原來貌似確實沒有讀完過……

2019-04-21 哲学的奥德赛 : 《王制》引论 [美]郝岚 / 李诚予 / 华夏出版社 / 2016-3 / 36  好像有點粗放，讀理想國的話……

2019-04-21 薄伽梵歌 [古印度] 毗耶娑 / 黄宝生 / 2011-6 / 20.00元  不知道爲什麼圖書館是把這個放在文學類的書記裏。

2019-04-25 正确的公司治理 马利克 / 朱健敏 / 2009-6 / 42.00元 算是應景讀的來着。

2019-04-26 拉丁美洲诗选 赵振江 / 云南人民出版社 / 1996-10 / 19.00  快速看完，說雲南人民出版社曾經如此強大？！

2019-04-26 雅各布·布克哈特 [德] 卡尔·洛维特 / 楚人 / 三辉图书/商务印书馆 / 2013-9-1 / 42.00  算是終於看完，其實應該是去年還是前年就買的書了。

2019-04-26 公司理财 [美] 斯蒂芬 A. 罗斯、[美] 伦道夫 W. 威斯特菲尔德、[美] 杰弗利 F. 杰富 / 吴世农、沈艺峰、王志强 等 / 机械工业出版社 / 2012-1 / 88.00元  準備再讀幾遍。

2019-04-27 国际结算 梁远辉 / 华中科技大学出版社（原华中理工） / 2007-1 / 29.50元  再次應景。

2019-04-28 白色的长椅 : 外国诗歌丛书 加贝 / 杨燕杰 / 上海译文出版社 / 1996-11 / 7.90￥ 其實挺好看的，說起來總是不那麼什麼集體主義？

2019-04-29 视觉新论 [英] 乔治·贝克莱 / 关文运 / 商务印书馆 / 2017-4 / 12.00元  算是讀書計劃內的，不過終於買到了這一本。雖然看得卻是有些草率。

然後又該如何繼續了呢。說起來好像也沒有很清晰的圖像。
