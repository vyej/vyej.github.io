2019-12-03 记忆 : 历史与政治理论 [美] 埃里克·沃格林 / 朱成明 / 华东师范大学出版社 / 2017-10 / 88 其實很用有。

2019-12-14 自传性反思 沃格林、桑多兹 / 徐志跃 / 华夏出版社 / 2009-08 / 22.00元  也不免八卦。

2019-12-14 历史理性批判手稿 (德)狄尔泰 / 陈锋 / 上海译文出版社 / 2012-7 / 25.00元 果然是手稿。

2019-12-18 论中国 [美]基辛格 / 胡利平、林华、杨韵琴、朱敬文 / 中信出版社 / 2012-10 / 68.00元  好像說的都沒有什麼錯。

2019-12-18 对莱布尼茨哲学的批评性解释 罗素 / 2011-5 / 46.00元  終於讀完，其實貌似也很有用。自帶某種羅素式的清晰。

2019-12-19 自由市场经济学 : 一个批判性的考察 安德鲁·肖特（Andrew Schotter） / 中国人民大学出版社 / 2013-4-1 / 35.00元 隨便看看好了。

2019-12-19 国际经济秩序的演变 [美] 阿瑟·刘易斯 / 乔依德 / 商务印书馆 / 2017-7 / 15 其實也很國貿。

2019-12-19 托兰德与激进启蒙 刘小枫 编 / 冯庆 等 / 华夏出版社 / 2015-1 / CNY 39.00 爲什麼總是想着的是托蘭德的帝國夢。

2019-12-19 基于内部资本市场的投融资问题研究 宁宇新 / 2010-3 / 23.00元 算是寫論文參考吧。

2019-12-21 伽达默尔论柏拉图 伽达默尔 / 余纪元 / 光明日报出版社 / 1992 / 5  果然是很老的書了。竟然沒有看過以前。

2019-12-21 历史的起源与目标 [德]雅斯贝斯 / 魏楚雄、俞新天 / 华夏出版社 / 1989 / 5.95 也是終於看完。其實還是很歷史的，而不是哲學的。

2019-12-29 亚历山大的克雷芒 [意] 塞尔瓦托·利拉 / 范明生 等 / 华夏出版社 / 2004-9 / 35.00元 這個應該也是受了沃格林影響，不然以前應該是看不下去的。不過現在看來還很有意思了。

2019-12-29 罗马史（第一卷）[德] 特奥多尔·蒙森 / 李稼年 译、李澍泖 校 / 商务印书馆 / 1994-10 / 12.00元 好吧，也算開始看羅馬史了。

合計13本書，其實想說的是這個閱讀的進度貌似有點不夠看了，要如何加快呢。起碼更有效率呀。
