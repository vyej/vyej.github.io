9月卻是過得很快，貌似還曾經有很多計劃。然後就沒有了下文。

9月讀書，大抵也是如此。其實好像計劃了很多，然後什麼也沒有做成，比如本來以爲可以完成韋伯文集的各種。

好吧，總共讀了11本，還有很多其實應該是重讀來着。

2020-09-01 自我分析纲要 [法] 皮埃尔·布迪厄 / 刘晖 / 中国人民大学出版社 / 2017-1-15 其實卻是算是自傳。只是重來不知道布迪厄的阿爾及利亞背景和做過阿隆的祕書。

2020-09-01 鲍勃.迪伦传 萨利.巴伯 / 王健 / 2016 內容少的可以算是假的書了吧。 

2020-09-02 论老年 论友谊 论责任 [古罗马] 西塞罗 / 徐奕春 / 商务印书馆 / 2003-12 / 18.00元  算是重讀？配合着施特勞斯講稿。

2020-09-03 韦伯作品集Ⅶ ：社会学的基本概念 （德）马克斯・韦伯 / 顾忠华 / 广西师范大学出版社 / 2005-03 / 10.00 好吧，還是跳過了中國宗教這個。

2020-09-05 经济理论和方法史（第四版） 小罗伯特.B.埃克伦德 / 张凤林 / 中国人民大学出版社 / 2001-1 / 56.00元 說起來也算終於讀完，都忘記了這個經濟史閱讀計劃了。

2020-09-07 批判施塔姆勒 [德] 马克斯·韦伯 / 李荣山 译、李康 校 / 上海人民出版社 / 2020-3 / 49.00元 以前竟然沒有看過？！好吧，算是讀完。在重新整理了一下所謂的閱讀韋伯之後。

2020-09-07 经济、诸社会领域及权力（第2卷） : 韦伯文选第二卷 韦伯(德) / 李强 / 三联书店 / 1998-05 / 7.80  這個是確實沒有看過……

2020-09-08 非正当性的支配 : 城市的类型学 （德）马克斯・韦伯 / 简惠美 / 广西师范大学出版社 / 2005-03 / 16.00元 好吧，算是繼續。

2020-09-10 论神性 (古罗马) 西塞罗 / 石敏敏 / 商务印书馆 / 2012-6 / 26.00元  其實也還好了，西塞羅的書還卻是很多。

2020-09-12 形而上学导论 海德格尔 / 2011-7 / 25.00元  這個確實是重讀了。

2020-09-30 西塞罗的政治哲学 [美]施特劳斯 讲疏、[美]尼科尔斯 编订 / 于璐 / 华东师范大学出版社 / 2018-6 / CNY 138.00 總算讀完。中間一段講自然的，引入各種亞里士多德卻是很海德格爾，不過貌似弄清楚了很多東西，而還沒有開始感慨海德格爾，又開始了所謂日常生活道德的解釋，貌似確實很日常，深有其感。像說凱恩斯一樣，大概也還是可以說，某種程度上，我們都是海德格爾主義者。

量還是不夠吧，還有英語閱讀呢，說起來，是因爲有找到了[書籍](https://1lib.net/)的網站，又可以每天愉快地下書了，所以就沒有看了……
