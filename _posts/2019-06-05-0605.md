也算是更新了blog，換一個地址？還是更換一下所謂的書寫的心態。更爲誠實和簡單一點？

2019年5月讀書情況：

2019-05-04 际贸易实务 黎孝先 编 / 人民大学 / 2004-1 / 26.00元   看教材。

2019-05-05 MATLAB Demystified David McMahon / McGraw-Hill Professional / 2007-04-06 / USD 21.95  其實看的是中文版。說確實把數學丟了呀。

2019-05-07 制度变迁的广义理论 唐世平 / 沈文松 译、唐世平 校 / 北京大学出版社 / 2016-10 / 35.00元  水。没有确定出发点的制度研究终于还是文献综述，所有都是变量最后迷失在变量中。

2019-05-07 公司治理与会计政策选择研究 王宁 著 / 中国书籍出版社 / 28元  圖書館翻完來着。

2019-05-07 生存哲学 [德]卡尔·雅斯贝斯(Karl Jaspers) / 王玖兴 / 上海译文出版社 / 2005.9 / 10.00元  讲哲学时候像海德格尔，讲现实时候像胡赛尔，讲宗教时候像舍勒，jaspers确实是一个时代的终结了。

2019-05-08 古兰经 马坚 / 中国社会科学出版社 / 1981年4月 / 3.00  中間很多一部分都不知道怎麼看完的。打瞌睡了。

2019-05-09 尤比克 [美]菲利普·迪克 / 金明 / 译林出版社 / 2013-9-1 / 26.00元  確實是嗑藥後寫出的。

2019-05-13 道德的谱系 [德]尼采（Friedrich Nietzsche） / 梁锡江 / 华东师范大学出版社 / 2015-5 / 39.80  好吧，好像說的都沒有怎麼錯。

2019-05-13 古典音乐说明书 [日] 饭尾洋一 / 刘雨轩 / 后浪丨北京联合出版公司 / 2017-6 / 36.00元  應該是現場古典音樂說明書。

2019-05-14 每周健身4小时 蒂莫西·费里斯 / 海绵 / 文化发展出版社 / 2017-6-1 / 48.00元  終於看完。其實作者貌似很羅嗦的。

2019-05-23 道德与哲学的修辞术 : 柏拉图的《高尔吉亚》和《斐德若》 [美]伯纳德特(Seth Benardete) / 赵柔柔、李松睿 / 华东师范大学出版社 / 2016-8-26 / 48.00  好吧，是有個Benardete計劃的吧。

5月其實沒有怎麼認真讀書。開始忙碌起來的樣子？說笑一點。還是被網絡小說耽誤。各種都有吧。

然後呢。也算已然半年。毫無進展。一路潰敗。

就這樣混下去了？說起來也是慚愧。還是不過都是自我安慰。又應該如何破呢。
