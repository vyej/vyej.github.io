其實8月也是困亂中過來，從內到外的混亂？所以結果就是買了很多本來不應該買的東西，比如跑步鞋和原子筆。

好吧，是因爲秋天的緣故麼。說起來，所以需要一點木頭的東西把玩一下。好吧，三菱確實漂亮，晨光還沒有收到，無印良品就有點性價比不高了？也不知道自己在說什麼。

2019-08-02 蜜汁炖鱿鱼 墨宝非宝 / 百花洲文艺出版社 / 2016-1 / 29.80元  8月從言情小說開始看起。貌似書沒有電視劇好看？雖然我也不看電視劇來着。

2019-08-05 论实证精神 [法] 奥古斯特·孔德 / 黄建华 / 商务印书馆 / 2011-7 / 18.00元 圖書館看來着，還是前面涂爾干說的，所以其實在惦記時不時應該看看聖西門。

2019-08-05 泛神论要义 : 或一个著名协会的诵文 (英) 约翰·托兰德 / 陈启伟 / 商务印书馆 / 2011-6 / 10.00元 托蘭德，這個確實不錯。只是可惜還有本沒有看完的經典與解釋。

2019-08-05 基督教并不神秘 (英) 约翰·托兰德 / 商务印书馆 / 2011-6 / 16.00元 繼續繼續。

2019-08-05 一个海难幸存者的故事 [哥伦比亚] 加西亚·马尔克斯 / 陶玉平 / 南海出版公司 / 2017-6 / 35.00元 作爲記者的馬爾克斯。

2019-08-05 早期基督教与希腊教化 [德]瓦纳尔·耶格尔 / 吴晓群 / 上海三联书店 / 2016-3 / 25.00元 莫名好看。

2019-08-05 给塞伦娜的信 [英] 约翰·托兰德 / 陈启伟 / 商务印书馆 / 2011-6 / 19.00元 還是托蘭德。

2019-08-08 Hear the Wind Sing 村上 春樹、アルフレッド・バーンバウム、Alfred Birnbaum / Kodansha / 1987-02 / JPY 7.14 算是今年看完的第一本英文書吧。終於。在計劃了8個月之後。

2019-08-13 符号帝国 罗兰巴特 / 孙乃修 / 商务印书馆 / 1994-2 / 8.20元 其實算是第一本羅蘭巴特吧，其實還挺看好。

2019-08-16 人类理智新论 : 全二册 莱布尼茨 / 陈修斋 / 商务印书馆 / 1982-11 / 50.00元  勉強讀完，說起來，是自己15年前買的書了吧。還在地壇書市。

2019-08-19 卡瓦菲斯诗集 [希腊] 卡瓦菲斯 / 黄灿然 / 重庆大学出版社/楚尘文化 / 2014-1-1 / 45.00 有點伊斯坦堡的味道。

2019-08-19 阿奎那政治著作选 (意) 阿奎那 / 马清槐 / 商务印书馆 / 2011-6 / 20.00元 好像節選本沒有看出什麼所以然來。

2019-08-19  山茶文具店 [日] 小川糸 / 王蕴洁 / 湖南文艺出版社 / 2018-3 / 49.80 這個就是開始買文具的原因麼。

2019-08-20 漂泊的荷兰人 : 歌剧歌词 瓦格纳 / 雨三 / 音乐出版社 / 1964-5 / 0.38元 2019-08-20 读过 看着劇本聽音樂來着，好像音樂沒有聽完。

2019-08-21 邂逅 : 后现代主义之后的历史哲学 [波兰]埃娃·多曼斯卡 / 彭刚 / 北京大学出版社 / 2007-12 / 35.00元 其實應該叫元史學週邊。

2019-08-22 自然与快乐 : 伊壁鸠鲁的哲学 [古希腊] 伊壁鸠鲁、[古罗马] 卢克来修 / 包利民、刘玉鹏、王纬纬 / 中国社会科学出版社 / 2004-11 / 20.00元 終於手機上讀完，也算不虧我的水墨屏手機呀。

2019-08-22 宗教的自然史 [英] 大卫·休谟 / 曾晓平 / 商务印书馆 / 2014-8 / 15.00 這是難得自己買了一本書。

2019-08-23 自然宗教对话录 休谟 / 2011-5 / 15.00元  算是繼續，都可以歸在康德計劃裏面麼，說這也是這幾個月的重點啊。

2019-08-28 古今自由主义 （美）列奥·施特劳斯 / 马志娟 / 江苏人民出版社 / 2012-6 / 32.00元 其實那些論文前面大都已經讀過。

2019-08-28 孔子家语 王国轩 / 中华书局 / 2011-3 / 33.00元 算是看完。論語計劃麼。接下來讀個荀子？

2019-08-30 人类理解论（套装上下册） [英] 洛克 / 关文运 / 商务印书馆 / 2011-6 / 90.00元 終於圖書館讀完。貌似其實對看萊布尼茨的那個的。可以還有休謨的薄薄的那本沒有讀完。

說來，終於在8月的時候，一個月讀書超過了20本？雖然大部分其實是圖書館集中讀的。

也是圖書館綜合徵，太多的書了，反而迷失了自己要做什麼和應該做什麼。說起來只是太多書都是自己想要的麼。也不免貪婪。

主要接下來應該完成的是什麼呢。

上面說的康德計劃，論語計劃，然後其實還有更重要的宏觀和會計？

所以，事情太多反而什麼都不想做。大抵如此。

------

說起來，甚至堅持每天寫點什麼都做不到。還有各種計劃呢。其實只是自己的詞彙貧乏吧。也是一說。

過於貧乏和貪婪。說起來好像還真的還是一件事情。然後終於拖延。
