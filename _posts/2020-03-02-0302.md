依然算是沒有計劃。雖然還是沒有完成的錢德勒，沒有完成的管理學和宏觀。

2月的閱讀就更是可憐了吧。讀書如下：

2020-02-08 荷马笔下的伦理 刘小枫 陈少明 主编 / 赵蓉 等 / 华夏出版社 / 2010-04 / 33.00元 總算看完了一本kindle裏面，ku借的書。

2020-02-19 论原因、本原与太一 [意] 乔尔丹诺·布鲁诺 / 汤侠声 / 商务印书馆 / 1998-1-1 / 25.00元 好吧，可以說是沃格林的緣故麼。

2020-02-21 一只狼在放哨 : 阿巴斯诗集 [伊朗] 阿巴斯·基阿鲁斯达米 / 黄灿然 / 中信出版集团 / 2017-7 / 39.00  讀書不夠，詩集來湊。

2020-02-22 水的空白 [罗]马林·索雷斯库 / 高兴 / 上海人民出版社 / 2013-6 / 35.00 繼續，如上。

2020-02-26 辩证法的历险 : 法国思想家译丛 [法]莫里斯·梅洛-庞蒂 / 杨大春、张尧均 / 上海译文出版社 / 200901 / 28.00元 怎麼覺得一直就是說，薩特傻逼，臭不要臉。

2020-02-28 新史学 詹姆斯·哈威·鲁滨孙 / 齐思和 / 商务印书馆 / 1964-6 / 18.00元 算是進展，這應該是梅尼克的同學吧。雖然是美國人。只可惜後面看得有點快。應該對照梅尼克來讀的。

2020-02-28 春盏 廖伟棠 / 四川文艺出版社 / 2016-6-1 / 48.00元  評分也不至於這麼低麼。就算人家拐走了院花師姐。

2020-02-29 恋人絮语 : 一个解构主义的文本[法]罗兰·巴特 / 汪耀进、武佩荣 / 上海人民出版社 / 2009-06-01 / 24.00元 一種活着某種或者很多種可能性。如是。

2月飽沾墨水。果然2月的閱讀很水。可能也是買了很多鋼筆的緣故？起碼應該有點羞愧之心麼。

也不知道接下來應該如何閱讀了。各種。

不過算是重新開始康德了，唯一一點積極的地方。
