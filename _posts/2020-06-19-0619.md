5月其實也沒有怎麼讀書。不知道自己在忙什麼。跑了100公里？

好吧，閱讀如下：

2020-05-05 剑桥美国经济史（第三卷） : 20世纪 斯坦利·L·恩格尔曼等 / 蔡挺、张林、李雅菁 / 中国人民大学出版社 / 2008-07-21 / 98.00元  其實很好看來着。可惜計劃的錢德勒還沒有繼續下去。

2020-05-07 死论 E.云格尔 / 林克 / 三联书店上海分店 / 1995-04 / 7.50  現代基督教的書看起來都想讀者風。

2020-05-09 康德著作全集（第8卷） : 1781年之后的论文 [德] 伊曼努尔·康德 / 李秋零 编译 / 中国人民大学出版社 / 2010 / 56.00元 好吧，算是繼續。大抵6月完成全部的可能也很低了。

2020-05-10 想见你 简奇峰、林欣慧 / 浙江人民出版社 / 2020-5 / 49.80  其實還是電視劇更多細節和回味。

2020-05-15 科学社会学（全二册） （美）R.K.默顿 / 鲁旭东/林聚任 / 商务印书馆 / 2003-11 / 51.00  貌似名氣很大。

2020-05-22 论法的精神 : （上下卷） [法]孟德斯鸠 / 许明龙 / 商务印书馆 / 2012-5 / 88.00元  清醒記，確實可以叫。要怎麼重讀來着。

好吧，一個月6本書，這個貌似太少，只是我做什麼去了呢。

說閱讀的熱情如何變成了焦慮。好像是一個不錯的問題。
