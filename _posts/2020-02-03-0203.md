2020年，說隔離其實也是閱讀的好時間，然並卵。

1月讀書如下：

2020-01-01 罗马史（第二卷） [德] 特奥多尔·蒙森 / 李稼年 译、李澍泖 校 / 商务印书馆 / 2004-7 / 13.00元  繼續羅馬史。

2020-01-03 罗马史（第三卷） [德] 特奥多尔·蒙森 / 李稼年 / 商务印书馆 / 2005-9 / 21.00元 繼續羅曼史。

2020-01-09 牛顿自然哲学著作选 [美]H.S.塞耶 / 上海外国自然科学哲学著作编译组译 / 上海人民出版社 / 1974年11月  應該也是歸在康德的計劃裏面來着？

2020-01-16 意象派诗选 [英]彼得琼斯 / 裘小龙 / 漓江出版社 / 1986-8 / 1.25元 其實還挺好看，還有八卦。

2020-01-17 闪闪发光的人生 [日]小川糸 / 吴曦 / 湖南文艺出版社 / 2019-1-31 / 48.00元 好吧，在文具控的路上越走越遠。開始惦記長刀研。

2020-01-23 卢梭·康德·歌德 [德] 卡西尔 / 刘 东 译 / 生活·读书·新知三联书店 / 2015-11 / 39.00 多年前，最早在席殊看到的小冊子，現在都賣了這麼貴了，還好看的是電子版。

2020-01-23 哲学与权力的谈判 (法) 德勒兹 / 刘汉全 / 译林出版社 / 2014-7-1 / 45.00 好吧，隨便一看。也算是看了多年的書終於看完了。

2020-01-23 埃克哈特大师文集 埃克哈特 / 荣震华 / 2011-6 / 56.00元  有點找不到感覺。

2020-01-29 政治观念史稿（卷五） : 宗教与现代性的兴起 [美] 埃里克·沃格林 / 霍伟岸 / 华东师范大学出版社 / 2009年8月 / 34.80元 羅馬史沒有了下文，只好沃格林了。

2020-01-31 革命与新科学 : 政治观念史稿·卷六 [美]沃格林（Voegelin,E.） / 谢华育 / 华东师范大学出版社 / 2009-8 / 29.80元 繼續沃格林。然後買了寫樂的標準魚雷，卻捨不得賣個全套的政治觀念史稿，也是醉了。不過主要是惦記着電子版的緣故？！


勉強，在過年期間，總共一月讀了10本書。貌似也是2020各種不順利的表現之一吧。

然後說，2月更困難。
