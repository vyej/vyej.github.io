好像最近以來，有段時間，這個readin的標籤都沒有和org一起了。說的是沒有計劃的閱讀。

反正困難時期什麼的說多了，好像也沒有了下文。

11月讀書也不多，唉。

2022-11-01 美的六种命名 [美] 克里斯平·萨特韦尔 / 郑从容 / 南京大学出版社 / 2017-5-1 / 38.00元 好像沒有什麼特別的。

2022-11-01 黑格尔的精神现象学 [德] 马丁·海德格尔 著、[德] 英格丽特·舒斯勒 编 / 赵卫国 / 南京大学出版社 / 2017-7-1 / 38.00元 總感覺以前就看過了。

2022-11-02 紧急中的冥想 [美]弗兰克·奥哈拉 / 李晖 / 北京联合出版公司 / 2021-3 / 52.00元  終於不是很基的美國詩人。

2022-11-05 生存神学与末世论 布尔特曼、等 / 李哲汇、朱雁冰 / 上海三联书店 / 1995-12 / 9.40  其實還聽好看了。

2022-11-18 利益集团与贸易政策 G.M.格罗斯曼 / 李增刚 / 中国人民大学出版社 / 2005-8 / 39.00元  如果沒有細看數學模型，是不是都不應該算看過了。

2022-11-19 傅山的交往和应酬（增订版） : 艺术社会史的一项个案研究 白谦慎 / 广西师范大学出版社 / 2016-3 / 54.00 只是按今天的話說，傅山就是情商很高啊。

2022-11-19 敦煌书法 秦川 / 清华大学出版社 / 2019 / 56.8  貌似紀錄片可能會更好看一些。

2022-11-21 例外状态 : 《神圣之人》二之一 [意] 吉奥乔·阿甘本 / 薛熙平 / 西北大学出版社 / 2015-1  還真的是失眠看完？雖然不脫離福柯的藩籬，但其實也很有趣了，相比其他的一些。

2022-11-21 灰烬的光辉 : 保罗·策兰诗选 [德]保罗·策兰 / 王家新 / 广西师范大学出版社 / 2021-1 / 82.00  好像沒有上一個看的譯本黑暗？

9本書，其實還真的是沒有看什麼。

貌似倒是看了兩個劇，這個事情啊。
