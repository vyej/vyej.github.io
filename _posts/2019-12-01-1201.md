其實也沒有都什麼，只是還有點偏科。

2019-11-05 历史意识的维度 [法] 雷蒙·阿隆（Raymond Aron） / 董子云 / 华东师范大学出版社 / 2017-1 / 58.00  竟然還有沒有讀過的雷蒙·阿龍，好吧，其實也確實有。

2019-11-06 风险与好的决策 [德]格尔德·吉仁泽 / 王晋 / 中信出版股份有限公司 / 2015-4 / 59.00元 好吧，通俗讀物。

2019-11-06  俄耳甫斯教祷歌 吴雅凌 编译 / 华夏出版社 / 2006-1 / 17.00元 這個倒是莫名好看。

2019-11-09 狄俄尼索斯颂歌 : 经典与解释. 尼采注疏集 (德)尼采 / 孟明 / 华东师范大学出版社 / 2013-8-1 / 48.00 好吧，其實就是尼采晚期詩集。

2019-11-09 阿赫玛托娃诗全集（第三卷） : 1958-1965 [俄] 阿赫玛托娃 / 晴朗李寒 / 人民文学出版社 / 2017-4 / 66.00 終於看完，原來全集也就是有三卷，以前一直以爲5卷還是6卷，一直心心念念的圖書館。

2019-11-10 荷马的竞赛 : 尼采古典语文学研究文稿选编 [德] 尼采 / 韩王韦 / 上海人民出版社 / 2018-3 / 42 也算是尼采作爲古典文學家的論文麼。其實瓦格納可能更欣賞的尼采。

2019-11-10 古希腊抒情诗集（全4册） [德] 恩斯特·狄尔 编 / 王扬 / 上海人民出版社 / 2018-9 / 288.00元 其實還是希臘文對照，不過好吧，單純作爲長見識的看好了，有沒有過一個學習希臘語的計劃呢。

2019-11-14 柏拉图对话中的神 : 薇依论古希腊文学 【法】西蒙娜·薇依、Simone Weil / 吴雅凌 / 华夏出版社 / 2012-10-1 / 45.00元 其實比較奇怪的是，爲什麼薇依一直在強調俄爾浦斯密儀什麼的，好像她知道什麼內幕的樣子，然後她倒是也是好的古典基礎呀。

2019-11-29 居鲁士的教育 : 色诺芬注疏集 [古希腊] 色诺芬 / 沈默 / 华夏出版社 / 2007-6 / 49.00元 其實要不看註釋，才能容易找到節奏，也是看得困難的樣子。

2019-11-30 哥特史 [拜占庭] 约达尼斯 / 罗三洋 / 商务印书馆 / 2011-8 / 28.00元 其實還挺好看。

2019-11-30 阿古利可拉传 日耳曼尼亚志 [古罗马] 塔西佗 / 马雍、傅正元 / 商务印书馆 / 2011-7 / 15.00元 微信讀書上半天看不完的日耳曼尼亞志，找本紙質的書一下就看完了。

2019-11-30 凯撒的剑与笔 多人 / 李世祥 编译 / 华夏出版社 / 2009-02 / 22.00元 算是對白天圖書館見到凱撒的書的疑問的解答吧。

算是勉強看了12本書。貌似如果要做學問的話，也是差了太多。所以，也是時候考慮或者改變了吧。
